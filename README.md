# AI Crypto Research Agent

An AI-powered multi-agent crypto research assistant built using GPT, Claude, and Gemini workflows.

Features:
- On-chain token analysis
- Scam/risk detection
- Wallet activity monitoring
- Multi-agent reasoning pipeline
- Automated market sentiment summaries
- Telegram/WhatsApp style alert workflows

Tech Stack:
- OpenCode/OpenClaw
- GPT / Claude / Gemini
- Python
- API integrations
- Crypto market data tools

Purpose:
Reduce manual crypto research time and improve scam detection accuracy through AI-driven analysis.
